# Screenshots

Add screenshots here after running the project locally.

## How to capture

1. Run `make up`
2. Run `make ingest TICKER=AAPL` in a second terminal
3. Open http://localhost:5173
4. Take screenshots of each view and save them here with these exact names:

| Filename | What to capture |
|---|---|
| `leaderboard.png` | Signal Leaderboard panel — top companies list |
| `sentiment_timeline.png` | Recharts line chart showing sentiment across call |
| `anomaly_cards.png` | Red/yellow anomaly cards on right panel |
| `transcript_viewer.png` | Scrollable transcript with colored sentences |
| `qa_box.png` | QA input box with a sample answer streaming |
| `k8s_pods.png` | Terminal output of `kubectl get pods -n earningsiq` |
